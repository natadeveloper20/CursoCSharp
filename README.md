## 🛠️ Requisitos

- [.NET SDK](https://dotnet.microsoft.com/download) instalado
- Visual Studio Code (o el IDE que prefieras)

## ▶️ Cómo correr los ejercicios

Desde la carpeta del ejercicio que quieras probar:

```bash
dotnet run
```

O abrí `CursoCSharp.sln` desde Visual Studio / VS Code para ver todo el proyecto en conjunto.
